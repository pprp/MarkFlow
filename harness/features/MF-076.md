# MF-076 - Paste as plain text shortcut strips rich formatting before insertion

## Notes
Implemented on 2026-04-15 in `packages/editor/src/editor/extensions/smartPaste.ts` by tracking a short-lived Cmd/Ctrl+Shift+V intent window so the next paste uses `text/plain` even when the clipboard also carries HTML, while preserving the default HTML→Markdown smart-paste path and existing image-paste priority. Focused coverage in `packages/editor/src/editor/__tests__/smartPaste.test.ts` now verifies default rich-text conversion, plain-text shortcut bypass, one-shot shortcut reset, and image-paste precedence. Sources: https://support.typora.io/Copy-Paste/ and https://support.typora.io/Shortcut-Keys/.

## Steps
- Copy rich-text content from an external source.
- Press Cmd/Ctrl+Shift+V in the editor and confirm the inserted text contains no markdown conversion and no HTML residue.
- Repeat with Cmd/Ctrl+V and confirm smart-paste conversion (MF-026) still applies on the default paste path.
