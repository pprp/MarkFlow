# MF-114 - Auto-update channel delivers signed beta builds on every main merge

## Notes
README roadmap commits to beta builds on every `main` merge with auto-update. Requires packaging (MF-113) and release signing to land first.

## Steps
- Wire `electron-updater` (or equivalent) into the desktop shell, pointed at the beta channel.
- Ensure every `main` merge publishes a signed beta artifact and release feed entry.
- Confirm an installed beta build detects a newer release and applies the update on relaunch.

## Automated Verification
- pnpm --filter @markflow/desktop build

## Manual Verification
- Install a prior beta, merge to main, and confirm the running app upgrades itself.
