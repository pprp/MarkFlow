# MF-162 - Outline panel supports header filtering, highlight-current, and flat/collapsible display modes

## Notes
Typora's Outline docs cover header filtering/search, highlighting the current header, and switching between flat and collapsible display modes. MarkFlow already showed heading hierarchy plus active-heading sync through MF-020 and MF-051, but it did not expose filter controls or alternative outline display modes. Sources: https://support.typora.io/Outline/
