# MF-162 - Outline panel supports header filtering, highlight-current, and flat/collapsible display modes

## Notes
Typora's Outline docs cover header filtering/search, highlighting the current header, and switching between flat and collapsible display modes. MarkFlow already shows heading hierarchy and live outline sync through MF-020 and MF-051, but it does not yet expose filter controls or alternative outline display modes. Sources: https://support.typora.io/Outline/

## Steps
- Open a document with multiple heading levels and enough entries to require filtering.
- Filter the outline and confirm only matching headings remain while click-to-jump still works.
- Toggle current-header highlighting and confirm the active section stays visually distinct as the editor selection moves.
- Switch between flat and collapsible outline modes and confirm the same headings remain navigable.

## Automated Verification
- Add editor/component tests for outline filtering, current-header highlighting, and flat/collapsible rendering.
- Add integration coverage proving click-to-jump and active-heading sync still work after filtering or mode changes.
- Run relevant editor tests and `pnpm harness:verify`.

## Manual Verification
