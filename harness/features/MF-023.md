# MF-023 - Mermaid and other diagram fences render live previews while preserving fenced source

## Notes
Fenced mermaid blocks render live SVG via Mermaid.js (lazy-loaded). Cursor-aware source reveal. SVG cache. CSS: .mf-mermaid.

## Steps
- Open fenced diagram blocks such as Mermaid flowcharts or sequence diagrams.
- Render the document in WYSIWYG mode and confirm a live diagram preview appears outside the active edit state.
- Edit the fenced source and verify the preview refreshes without mutating or losing the original markdown block.
