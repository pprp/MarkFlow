# MF-140 - Markdown auto-pair wraps selections and completes Typora-style inline delimiters

## Notes
Typora supports preference-controlled auto pair behavior for normal brackets, quotes, and common markdown syntax. MarkFlow currently covers standard editor bracket closing plus some single-character markdown delimiters, but the backlog did not explicitly cover Typora-style completion and wrapping for inline code, math, superscript/subscript, highlight, and related markdown delimiters. Sources: https://support.typora.io/Auto-Pair/ and https://support.typora.io/Markdown-Reference/.

## Steps
- Type a supported inline markdown delimiter with an empty selection and confirm MarkFlow inserts the matching closing delimiter with the caret between the pair.
- Select inline text and type a supported delimiter, then confirm the selected text is wrapped and remains selected inside the pair.
- Confirm list bullets, identifiers, and ordinary punctuation entry are not intercepted by markdown auto-pair handling.
