# MF-150 - Citation manager inserts BibTeX and CSL-backed references without breaking markdown source

## Notes
Academics are a core MarkFlow audience, and citation support is a high-leverage writing feature. This feature introduces a source-preserving citation workflow: attach a bibliography file, search references, insert markdown citation keys, render readable citation chips, and feed bibliography metadata into export.

## Steps
- Let a workspace or document attach a BibTeX or CSL JSON bibliography source.
- Provide a citation picker that searches author, title, year, and citation key.
- Insert markdown-compatible citation syntax while rendering a compact citation chip away from the caret.
- Pass bibliography and CSL options through the export pipeline when supported.

## Automated Verification
- Add parser tests for BibTeX or CSL JSON reference ingestion and citation-key lookup.
- Add editor decoration tests proving citation chips preserve source text and reveal raw syntax at the caret.
- Add export tests proving bibliography metadata is passed to supported export targets without affecting unsupported exports.

## Manual Verification
- Attach a small bibliography, insert citations into a document, export to DOCX or PDF, and confirm the citations remain readable and source-safe.
