# MF-017 - Opt-in Yjs collaboration can bootstrap a shared document session

## Notes
Implemented as opt-in: yjs and y-codemirror.next added to @markflow/editor deps. packages/editor/src/editor/extensions/yCollab.ts exports yCollabExtension (binds a Y.Doc to CodeMirror via y-codemirror.next) and mergeYDocs (exchange state vectors for test/manual sync). Four CRDT convergence unit tests verify: single-peer sync, concurrent inserts from two offline peers, concurrent delete+insert, and markdown source preservation on no-op sync. Network provider (y-websocket) integration is left as a follow-up; current automated coverage exercises the full CRDT merge path without network.

## Steps
- Create a shared document provider.
- Join two editor clients to the same document.
- Confirm edits converge without source corruption.
