# MF-087 - Flowchart, sequence, gantt, Venn, and Ishikawa fences render alongside Mermaid

## Notes
Typora supports flowchart.js, js-sequence-diagrams, and Mermaid in fenced blocks. Sources: https://support.typora.io/Draw-Diagrams-With-Markdown/ and https://support.typora.io/What%27s-New-1.0/.
Mermaid 11.14.0 includes `venn-beta` and `ishikawa` diagram detectors, so MarkFlow only needs to pass those sources through the existing `mermaid` fence decoration/render pipeline.

## Steps
- Author fenced blocks with languages `flow`, `sequence`, `mermaid` (sequence/gantt/Venn/Ishikawa), and confirm each renders an inline diagram preview.
- Introduce a syntax error in one block and confirm the editor surfaces a localized render error without breaking siblings.
- Edit the source and confirm the diagram re-renders on next idle without duplicate DOM nodes.
