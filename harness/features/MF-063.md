# MF-063 - Minimap shows a scaled-down glyph view of the full document and supports click-to-scroll

## Notes
Implemented on 2026-04-16 in `packages/editor/src/components/Minimap.tsx`, `packages/editor/src/editor/MarkFlowEditor.tsx`, `packages/editor/src/App.tsx`, `packages/editor/src/styles/global.css`, `packages/editor/src/components/minimap.test.tsx`, `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx`, `packages/editor/src/__tests__/App.test.tsx`, `packages/desktop/src/main/menu.ts`, `packages/desktop/src/main/menu.test.ts`, and `packages/shared/src/index.ts`. The View menu now exposes `Toggle Minimap`, the editor renders a right-side canvas minimap that rasterises one row per document line into compact glyph bars, clicks map back to proportional line navigation, and a translucent viewport box tracks live scroll metrics from the active CodeMirror scroller. Automated verification passed on 2026-04-16 with the listed editor/desktop/shared commands plus `pnpm harness:verify`. `passes` remains false because this terminal session did not truthfully complete the required live visual check on a 1,000-line document.

## Steps
- Enable minimap from View menu.
- Confirm a right-side minimap renders a proportional representation of the document.
- Click a region of the minimap and confirm the editor scrolls proportionally.
- Confirm the viewport highlight box tracks scrolling.
