# MF-058 - Column (block) selection mode lets users select a rectangular region across multiple lines

## Notes
Implemented on 2026-04-23 in `packages/editor/src/editor/MarkFlowEditor.tsx` with CodeMirror's native `rectangularSelection` extension. The gesture is intentionally gated to Alt/Option+Shift+left-drag so the existing Alt-click multi-cursor behavior remains available.

## Steps
- Alt+Shift+drag to create a rectangular selection across 5 lines.
- Type a replacement string and confirm it replaces the selected column on every line.
- Confirm undo restores all lines atomically.
