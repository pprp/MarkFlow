# MF-008 - Save and Save As write the live editor buffer rather than the starter template

## Notes
Desktop menu actions now route through the renderer so save operations always use the live CodeMirror buffer; covered by `packages/editor/src/__tests__/App.test.tsx`.

## Steps
- Open or create a document and make edits in the editor.
- Use Save and Save As from the menu or renderer commands.
- Confirm the written file contents match the current CodeMirror document.

## Automated Verification
- pnpm --filter @markflow/editor test:run

## Manual Verification
- Save a file, reopen it, and confirm the latest edits persist.
