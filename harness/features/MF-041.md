# MF-041 - Vault sidebar shows a file tree for the active folder and supports open, rename, and delete

## Notes
Verified minimum file-sidebar parity: MarkFlow can open a folder, show a file tree, and perform open/rename/delete actions from the sidebar. Typora's current file-management docs still describe broader folder-side capabilities that remain outside this verified slice, including File List view, auto/manual refresh, sorting/grouping controls, drag-move, recent-location pinning, and open-in-new-window behavior. Sources: https://support.typora.io/File-Management/ and https://support.typora.io/Quick-Start/.

## Steps
- Open a folder as a vault from File > Open Folder and confirm a sidebar tree panel appears listing all markdown and asset files.
- Click a file in the sidebar and confirm the editor opens it, replacing or adding a tab for the new document.
- Rename and delete files from the sidebar context menu and confirm the filesystem reflects the changes.
