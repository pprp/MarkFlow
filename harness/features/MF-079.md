# MF-079 - Duplicate line or selection shortcut inserts a copy below without touching the clipboard

## Notes
Verified on 2026-04-16 against the implementation now present in `packages/editor/src/editor/extensions/smartInput.ts`, which now binds duplication to `Shift-Alt-ArrowDown` so `MF-101` can reclaim Typora's documented `Mod-Shift-D` delete-word shortcut without losing duplicate-line coverage. The command still duplicates the covered line block directly below without touching the clipboard, lands a single caret on the duplicate, preserves block selection on duplicated multi-line ranges, and isolates the change into one undo step. Covered by `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx`. `passes` remains false because the required live code-fence duplication check was not completed in this terminal session. Sources: https://support.typora.io/Shortcut-Keys/ and the `MF-101` prerequisite shortcut-conflict fix recorded on 2026-04-16.

## Steps
- Place caret on a line and press Cmd/Ctrl+Shift+D — confirm the line is duplicated directly beneath and the caret lands on the copy.
- Select multiple lines and repeat — confirm the block duplicates as a contiguous copy.
- Confirm undo collapses the duplication into a single atomic step.
