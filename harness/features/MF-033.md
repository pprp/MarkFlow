# MF-033 - Open Quickly fuzzy-searches files in the current folder and recent locations

## Notes
Implemented with `packages/editor/src/components/QuickOpen.tsx`, `packages/editor/src/App.tsx`, and `packages/desktop/src/main/fileManager.ts`, which build a Quick Open overlay from the current document folder plus recent files tracked by the desktop shell. This verified slice matches Typora's core quick-open workflow for current-folder and recent-file lookup; pinned folders/locations remain outside the shipped scope. Sources: https://support.typora.io/Search/ and https://support.typora.io/File-Management/.

## Steps
- Open a markdown file from a folder containing multiple documents, then press `Cmd`+`Shift`+`O` on macOS or `Ctrl`+`P` on Windows/Linux.
- Type part of a filename and confirm the quick-open overlay fuzzy-filters files from the current folder plus recent files.
- Select a result with the keyboard and confirm the target file opens without disrupting the current editor session.
