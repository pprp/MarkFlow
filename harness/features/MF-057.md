# MF-057 - Multi-cursor editing allows placing cursors on non-contiguous lines and editing them simultaneously

## Notes
Implemented on 2026-04-12 in `packages/editor/src/editor/MarkFlowEditor.tsx` by enabling `EditorState.allowMultipleSelections`, remapping `EditorView.clickAddsSelectionRange` to `Alt` so add-cursor clicks match the feature spec, and adding an explicit `Escape` keybinding that collapses back to the primary selection. `Mod-D` select-next-occurrence support now works through the existing CodeMirror `searchKeymap` because multiple selections are enabled. Automated coverage lives in `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx` and passed on 2026-04-12 with `pnpm --filter @markflow/editor test:run -- --grep multi-cursor`, plus `pnpm --filter @markflow/editor build` and `pnpm --filter @markflow/editor lint`. `passes` remains false only because the manual Alt-click interaction check is still pending.

## Steps
- Alt-click to place additional cursors on three separate lines.
- Type text and confirm it is inserted at every cursor simultaneously.
- Press Escape and confirm cursors collapse to a single primary caret.
