# MF-078 - Move line and selection up/down shortcut reorders blocks without clipboard usage

## Notes
Implemented and tightened on 2026-04-15. Typora 1.11 documents Alt+Up/Down for moving paragraphs or rows. MarkFlow inherits CodeMirror's default `Alt-ArrowUp` / `Alt-ArrowDown` `moveLineUp` and `moveLineDown` bindings through `packages/editor/src/editor/MarkFlowEditor.tsx`, and `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx` directly covers single-line moves, contiguous block moves in both directions, undo/redo atomicity, and real numbered markdown list item block reordering with preserved list structure. Final automated verification passed on 2026-04-22T18:08:42+0800 with `pnpm --filter @markflow/editor exec vitest run src/editor/__tests__/MarkFlowEditor.test.tsx -t "moves a numbered list item block"`, `pnpm --filter @markflow/editor exec vitest run src/editor/__tests__/MarkFlowEditor.test.tsx`, `pnpm --filter @markflow/editor lint -- src/editor/MarkFlowEditor.tsx src/editor/__tests__/MarkFlowEditor.test.tsx`, and `pnpm --filter @markflow/editor build`. Sources: https://support.typora.io/Shortcut-Keys/ and https://support.typora.io/What%27s-New-1.11/.

## Steps
- Place the caret on a line and press Alt+Up or Alt+Down — confirm the line swaps with the adjacent line while the caret stays on the moved line.
- Select three lines and press Alt+Up or Alt+Down — confirm the selection moves as a contiguous block while preserving the moved selection state.
- Verify undo/redo returns the document to the prior state atomically.
