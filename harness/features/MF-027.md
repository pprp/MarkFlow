# MF-027 - Word count surfaces live document and selection stats without treating markdown syntax as words

## Notes
Typora exposes live word count plus lines, characters, reading time, and selection counts, and explicitly excludes formatting-only markdown syntax from word totals. MarkFlow currently has no count surface or metric derivation in the editor UI. Sources: https://support.typora.io/Word-Count/ and https://support.typora.io/Quick-Start/.

## Steps
- Open a markdown document with prose, lists, and code, then observe a live word-count surface in the editor chrome.
- Edit the document and confirm word, line, character, and reading-time stats update continuously.
- Select a range and confirm the selection count is shown separately while markdown formatting syntax is excluded from word totals.

## Automated Verification
- Add derived-metric tests for document and selection counts, excluding formatting-only markdown syntax from word totals.

## Manual Verification
- Verify counts on mixed prose and markdown syntax match the detailed count surface for both whole-document and selected text.
