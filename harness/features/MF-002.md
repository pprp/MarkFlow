# MF-002 - Inline emphasis, inline code, and strikethrough hide syntax away from the caret

## Notes
Covered by `packages/editor/src/editor/__tests__/inlineDecorations.test.tsx`.

## Steps
- Type markdown with bold, italic, inline code, and strikethrough markers.
- Move the caret out of each formatted span.
- Confirm the rendered text appears without changing the underlying source string.

## Automated Verification
- pnpm --filter @markflow/editor test:run

## Manual Verification
- Check that formatted inline spans still reveal raw markdown when edited directly.
