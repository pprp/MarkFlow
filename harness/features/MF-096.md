# MF-096 - Custom keybinding editor lets users remap any palette-registered command

## Notes
Typora allows customizing shortcuts via a JSON file and UI surface. Sources: https://support.typora.io/Shortcut-Keys/ and https://support.typora.io/Custom-Shortcut-Keys/.

## Steps
- Open the Shortcuts pane in Preferences and search for a command.
- Record a new key chord and confirm it persists, conflicts are highlighted, and the prior binding is cleared.
- Reset an individual binding and confirm the default returns without affecting other custom bindings.

## Automated Verification
- Add tests for binding storage, conflict detection, and reset semantics.

## Manual Verification
- Rebind three commands, relaunch, and confirm the new bindings fire.
