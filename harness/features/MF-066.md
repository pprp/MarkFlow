# MF-066 - Style shortcuts wrap the current selection for bold, italic, underline, and links without leaving the editor flow

## Notes
Implemented on 2026-04-12. Added `Mod-b` (bold), `Mod-i` (italic), `Mod-u` (underline), and `Mod-k` (link) keyboard shortcuts to `packages/editor/src/editor/extensions/smartInput.ts`. Each shortcut wraps the selected text with the appropriate markdown wrapper, or inserts a wrapper with placeholder text and selects the content when the cursor has no selection. Link shortcuts scaffold `[text](url)` with the cursor positioned on the URL. Automated coverage lives in `packages/editor/src/editor/__tests__/smartInput.test.ts` (direct handler tests) and `MarkFlowEditor.test.tsx` (component-level bold and link shortcut tests). All 217 tests pass.

## Steps
- Select inline prose in the editor, press Typora's Bold and Italic shortcuts (`Cmd`/`Ctrl`+`B` and `Cmd`/`Ctrl`+`I`), and confirm only the selected range is wrapped in `**...**` or `*...*` source while the rendered styling appears again after the caret leaves the span.
- Select text, press Typora's Underline shortcut (`Cmd`/`Ctrl`+`U`), and confirm MarkFlow wraps just that range in `<u>...</u>` without corrupting adjacent markdown or leaving the editor.
- Select text, press Typora's Link shortcut (`Cmd`/`Ctrl`+`K`), and confirm MarkFlow scaffolds `[selected text](url)` around the selection so the existing link rendering/navigation path can take over once the destination is edited.
