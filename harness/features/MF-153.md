# MF-153 - Safe raw HTML style and sizing attributes survive WYSIWYG rendering

## Notes
Typora documents raw HTML support and image sizing through HTML attributes and inline styling in Markdown documents. MarkFlow currently preserves a safe tag subset from MF-038/MF-097, but generic inline style rendering is still stripped, so styled raw HTML does not match Typora's visual behavior. Sources: https://support.typora.io/HTML/ and https://support.typora.io/Resize-Image/.

## Steps
- Create markdown containing inline HTML such as `<span style="color:red">red text</span>` and safe sizing/style attributes.
- Render in WYSIWYG mode.
- Confirm supported style attributes affect only the intended HTML content.
- Confirm unsafe attributes, URL-bearing CSS, script tags, and event handlers are stripped or inert.
