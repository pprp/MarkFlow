# MF-045 - Global search supports case, whole-word, regex, and tag search across files

## Notes
Typora's Search and File Management docs describe sidebar-backed global search across files, including case-sensitive, whole-word, and regex toggles, plus literal `#tag` discovery across the opened folder. MarkFlow now exposes those match options in global search and sends them through the desktop search API while preserving the default literal, case-insensitive search path for existing callers and `#tag` tokens. Sources: https://support.typora.io/Search/ and https://support.typora.io/File-Management/.

## Steps
- Open a folder containing multiple markdown files and invoke global search from the sidebar or with `Cmd`/`Ctrl`+`Shift`+`F`.
- Search for a phrase, whole word, regex, or `#tag`-style token and confirm matching files plus snippet locations appear across the folder.
- Select a result and confirm MarkFlow opens the correct file, scrolls to the match, and highlights the selected occurrence.
