# MF-045 - Global search queries content across files in the opened folder and jumps to exact matches

## Notes
Typora's Search and File Management docs describe sidebar-backed global search across files, including case-sensitive, whole-word, and regex toggles, plus literal `#tag` discovery across the opened folder. MarkFlow currently only ships current-document find/replace under MF-028; repo search shows no folder index, sidebar search UI, or cross-file result navigation, so this is a genuinely missing Typora capability. Sources: https://support.typora.io/Search/ and https://support.typora.io/File-Management/.

## Steps
- Open a folder containing multiple markdown files and invoke global search from the sidebar or with `Cmd`/`Ctrl`+`Shift`+`F`.
- Search for a phrase, regex, or `#tag`-style token and confirm matching files plus snippet locations appear across the folder.
- Select a result and confirm MarkFlow opens the correct file, scrolls to the match, and highlights the selected occurrence.

## Automated Verification
- Add desktop and renderer integration tests for folder indexing, cross-file search filters (case, whole-word, regex), and result navigation into the correct file and text range.

## Manual Verification
- Open a folder with matches in several files, run global search from the sidebar, and confirm selecting results jumps to the right file and text range.
