# MF-089 - UI localization supports switching interface language at runtime

## Notes
Typora release notes highlight expanding UI language coverage. Sources: https://support.typora.io/What%27s-New-1.13/ and https://support.typora.io/Preferences/.

## Steps
- Expose a language selector in Preferences with at least English and one additional locale (e.g. 简体中文).
- Switch the locale and confirm menus, toolbars, dialogs, and palette entries retranslate without requiring a restart.
- Confirm document content is never translated and that per-user locale persists across launches.

## Automated Verification
- Add i18n resolver tests covering fallback chains and missing-key behavior.

## Manual Verification
- Switch between three locales and confirm full UI coverage plus right-to-left layout sanity if applicable.
