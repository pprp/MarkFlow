# MF-159 - Rendered diagram context actions copy and save generated SVG output

## Notes
Typora documents rendered diagram context actions for copying and saving generated diagram output. MarkFlow currently renders Typora-supported flow, sequence, and Mermaid fences, but the rendered diagram widget does not expose a copy/save action for the generated SVG. Sources: https://support.typora.io/Draw-Diagrams-With-Markdown/ and https://support.typora.io/Markdown-Reference/.

## Steps
- Open a document with a rendered Mermaid, sequence, or flow diagram.
- Use the rendered diagram action surface to copy the generated SVG output without selecting markdown source fences.
- Use the rendered diagram action surface to save the generated SVG output with a diagram-specific filename.
- Confirm failed or still-loading diagrams do not offer stale copy/save output.
