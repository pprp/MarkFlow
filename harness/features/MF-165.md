# MF-165 - HTML/PDF export preserves Typora-style outline options, including HTML outline panel and PDF document outline

## Notes
Typora's export docs say exported HTML can optionally include an outline panel, and exported PDFs auto-generate a document outline that appears in PDF readers. The same docs also say exported HTML outline behavior follows the user's current flat-vs-collapsible outline setting. MarkFlow already has partial groundwork through `MF-107` export support and the in-progress `MF-162` outline display modes, but it does not yet have a verified feature covering exported HTML outline inclusion or parity between in-app outline mode and exported outline behavior.

Sources:
- https://support.typora.io/Export/
- https://support.typora.io/Outline/

## Steps
- Export a document to HTML with outline inclusion enabled and confirm the exported file includes a navigable outline panel.
- Switch the in-app outline between flat and collapsible modes, export HTML again, and confirm the exported outline follows the active mode.
- Export the same document to PDF and confirm a document outline appears in the PDF reader's table-of-contents/sidebar UI.
