# MF-021 - A `[toc]` block expands into a live table of contents that stays in sync with headings

## Notes
Now uses extractOutlineHeadings from outline.ts for parser-backed heading extraction. Handles ATX and setext headings, de-duplicates anchors, and excludes fenced-code pseudo-headings. TOC anchor elements use class=mf-link so modifier-click navigation is handled by the editor. Covered by packages/editor/src/editor/__tests__/tocDecoration.test.tsx (7 tests).

## Steps
- Insert `[toc]` into a markdown document, press `Return`, and confirm it becomes a rendered table of contents block in the document.
- Add, remove, and rename headings across the document.
- Confirm the TOC updates automatically, excludes fenced-code pseudo-headings, preserves duplicate-heading anchor targets, and its entries navigate to the correct sections.
