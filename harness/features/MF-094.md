# MF-094 - Insert date/time action writes the current timestamp in user-selected format

## Notes
Typora includes configurable date/time insertion via shortcuts. Sources: https://support.typora.io/Shortcut-Keys/ and https://support.typora.io/Preferences/.

## Steps
- Invoke Insert Date / Insert Time from the palette or shortcut.
- Confirm the inserted string respects the format configured in Preferences (ISO 8601 default) and the user's locale/timezone.
- Confirm the action respects current selection: replacing it rather than appending.

## Automated Verification
- Add an editor test covering date/time insertion with stubbed `Date.now()` and a custom format string.

## Manual Verification
- Insert timestamps under two different format preferences and confirm output matches.
