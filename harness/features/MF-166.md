# MF-166 - Paragraph alert command converts selected paragraphs into GitHub-style callouts

## Notes
Typora 1.10 added a Paragraph -> Alert command that converts selected paragraphs into GitHub Alert blocks using the documented callout syntax. MarkFlow has callout rendering coverage, but the ledger did not yet track the editor command path for converting one or more selected paragraphs into alert blocks.

Sources:
- https://support.typora.io/What%27s-New-1.10/
- https://support.typora.io/Markdown-Reference/

## Steps
- Select one or more plain paragraphs in the editor.
- Run the alert paragraph command.
- Confirm each selected paragraph becomes a quoted GitHub-style alert block, for example `> [!NOTE]` followed by quoted body lines.
- Undo once and confirm the original paragraphs are restored.

## Automated Verification
- Add editor command/keymap tests that convert one selected paragraph and a multi-paragraph selection into alert blocks.
- Verify undo restores the original source text in one step.

## Manual Verification
- In the desktop editor, select several paragraphs and run the alert command from the menu or command palette. Confirm the source text and rendered callout match Typora's GitHub Alert behavior.
