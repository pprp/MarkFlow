# MF-099 - Auto pair brackets, quotes, and markdown delimiters wrap selections and preserve typing context

## Notes
Verified on 2026-04-15. `packages/editor/src/editor/extensions/smartInput.ts` already covered structural pairs plus context-aware `*`/`_` pairing and atomic backspace; this run expanded `packages/editor/src/editor/__tests__/smartInput.test.ts` to cover structural-pair selection wrapping and atomic deletion alongside the existing prose, unordered-list, and `snake_case` typing-context checks, so the listed automated commands fully prove the shipped scope. Typora's broader auto-pair parity for `~`, `=`, `$`, `^`, and a preference toggle remain separate gaps outside MF-099. Sources: https://support.typora.io/Auto-Pair/ and https://support.typora.io/Markdown-Reference/.

## Steps
- Type structural delimiters such as `(`, `[`, `{`, `"`, `'`, and backticks in the editor and confirm MarkFlow inserts the matching closer with the caret placed inside.
- Select a word and type `(`, `*`, or `_`; confirm the selection is wrapped rather than replaced, and typing `*` at the start of a line still begins a list while `_` inside identifiers remains literal text.
- Press Backspace inside an empty auto-inserted pair and confirm both sides are removed atomically without disturbing surrounding markdown.
