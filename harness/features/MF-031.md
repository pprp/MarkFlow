# MF-031 - Smart typography auto-converts ASCII punctuation to typographic equivalents as the user types

## Notes
Typora enables smart quotes and smart dashes under Preferences > Editor. Implementing this as an additional CodeMirror transaction filter in `smartInput.ts` alongside the existing auto-pair logic keeps the behavior opt-in per settings. Critical for academics and technical writers aiming for publication-quality output.

## Steps
- Type straight double and single quotes in prose context and confirm they become curly/smart quotes.
- Type `--` and `---` in prose and confirm they become en-dash and em-dash respectively.
- Verify that straight quotes and hyphens inside code spans and fenced blocks are never transformed.

## Automated Verification
- Add `smartInput.test.ts` cases covering quote-pairing substitution, dash ligatures, and code-range exclusion.

## Manual Verification
