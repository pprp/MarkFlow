# MF-160 - VS Code and Cursor can open markdown files directly in MarkFlow

## Notes
Typora 1.13 introduced an official "Open in Typora" extension for VS Code and Cursor, with explorer/editor-tab context menu actions that launch selected Markdown files in Typora. MarkFlow currently supports desktop launch targets and command-line style startup behavior, but it does not provide an IDE handoff extension or documented protocol for opening Markdown-like files from VS Code or Cursor into MarkFlow. Sources: https://support.typora.io/What%27s-New-1.13/ and https://marketplace.visualstudio.com/items?itemName=typora.typora-open-in-typora.

## Steps
- Install the MarkFlow IDE handoff integration in VS Code or Cursor.
- Right-click a supported Markdown-like file in the explorer and choose "Open in MarkFlow".
- Right-click an open editor tab for a supported Markdown-like file and choose "Open in MarkFlow".
- Confirm MarkFlow opens the exact local file path without creating a duplicate untitled tab.
- Confirm unsupported extensions do not show the action, or show a clear disabled/error state.

## Automated Verification
- Add extension or launch-target tests proving supported extensions are detected and unsupported extensions are rejected.
