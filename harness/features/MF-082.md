# MF-082 - Preferences dialog exposes editor, theme, export, and shortcut settings

## Notes
Typora ships a multi-tab Preferences window. Source: https://support.typora.io/Preferences/.

## Steps
- Open Preferences (Cmd/Ctrl+,) and confirm sections for Editor, Appearance, Export, and Shortcuts exist.
- Change a representative setting from each section, close the dialog, and confirm the change persists across restart.
- Reset to defaults and confirm every section returns to baseline without corrupting user content.

## Automated Verification
- Add desktop tests around preferences persistence (JSON read/write, migration, default reset).

## Manual Verification
- Walk through every preferences tab and verify each control is wired to real behavior.
