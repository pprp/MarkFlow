# MF-053 - Fuzzy document-wide search highlights all matches and jumps between them with keyboard shortcuts

## Notes
Run search in a Web Worker using a regex; stream decoration ranges back to viewport-visible lines first.

## Update 2026-04-19
- Implemented a dedicated `Cmd/Ctrl+F` document-search bar in `packages/editor/src/components/DocumentSearch.tsx`.
- The bar drives fuzzy document search through a line-bounded regex compiled in `packages/editor/src/editor/documentSearch.ts`.
- Match counting runs asynchronously through `packages/editor/src/editor/documentSearch.worker.ts`, with a main-thread fallback for test environments without `Worker`.
- Visible match highlighting and `Enter` / `Shift+Enter` navigation now live in `packages/editor/src/editor/MarkFlowEditor.tsx`, limited to viewport-visible lines to stay aligned with MF-048 virtual rendering constraints.
- App-level integration and regression coverage were added in:
  - `packages/editor/src/editor/__tests__/documentSearch.test.ts`
  - `packages/editor/src/__tests__/App.test.tsx`
- Automated verification completed:
  - `pnpm --filter @markflow/editor test:run -- --grep fuzzy-search`
  - `pnpm --filter @markflow/editor lint`
  - `pnpm --filter @markflow/editor build`
  - `pnpm harness:verify`
- Manual verification is still pending in this environment, so `harness/feature-ledger.json` remains unchanged.

## Steps
- Press Cmd+F to open the search bar.
- Type a query and confirm all matches are highlighted within 200 ms on a 180k-line file.
- Press Enter / Shift+Enter to cycle through matches.
- Confirm match count badge is accurate.
