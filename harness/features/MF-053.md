# MF-053 - Fuzzy document-wide search highlights all matches and jumps between them with keyboard shortcuts

## Notes
Run search in a Web Worker using a regex; stream decoration ranges back to viewport-visible lines first.

## Steps
- Press Cmd+F to open the search bar.
- Type a query and confirm all matches are highlighted within 200 ms on a 180k-line file.
- Press Enter / Shift+Enter to cycle through matches.
- Confirm match count badge is accurate.

## Automated Verification
- pnpm --filter @markflow/editor test:run -- --grep fuzzy-search

## Manual Verification
- Search for a common word in the large-file fixture; count highlighted matches.
