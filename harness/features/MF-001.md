# MF-001 - Headings render in WYSIWYG mode without mutating the markdown source

## Notes
Covered by `packages/editor/src/editor/__tests__/inlineDecorations.test.tsx`.

## Steps
- Load a markdown document that starts with `#` or `##` headings.
- Move the caret away from the heading line in WYSIWYG mode.
- Confirm the rendered heading is visible while the underlying source text remains unchanged.

## Automated Verification
- pnpm --filter @markflow/editor test:run

## Manual Verification
- Open the editor and move the caret in and out of heading lines.
