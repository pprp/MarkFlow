# MF-136 - Double-clicking an image opens a moveable floating lightbox window

## Notes
When a user double-clicks an image in the editor (WYSIWYG or reading mode), a floating lightbox overlay appears showing the full image. The overlay is draggable, closeable via Escape or clicking outside, and does not disturb editor state.

Implementation: add a `dblclick` handler in `packages/editor/src/editor/decorations/linkDecoration.ts` on the rendered `<img>` element. The handler creates (or re-uses) a singleton `<dialog>` or absolutely-positioned `<div>` appended to `document.body` containing `<img src="...">`, with pointer-event drag logic using `setPointerCapture`.

## Steps
- Open a markdown document containing an image (`![alt](url)`).
- Switch to WYSIWYG or Reading mode so the image renders.
- Double-click the image.
- Confirm a floating window appears with the image at full (or natural) size.
- Drag the window by its title bar to a new position and confirm it moves.
- Press Escape or click outside the window and confirm it closes.
- Confirm no editor content was modified.
