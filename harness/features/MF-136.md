# MF-136 - Double-clicking an image opens a moveable floating lightbox window

## Notes
Implemented in `packages/editor/src/editor/decorations/linkDecoration.ts`, using the existing rendered image widget path so double-clicking an image opens a singleton floating lightbox without mutating the markdown source. The lightbox reuses the committed `.mf-lightbox` styling, supports title-bar drag movement through pointer capture, closes on `Escape`, and now also closes on outside pointer presses while cleaning up the active global listeners.

Verified on 2026-04-25 with focused DOM coverage in `packages/editor/src/editor/__tests__/linkDecoration.test.tsx`. The added regression exercises the full MF-136 behavior surface on a rendered local image: double-click open, correct resolved `src`, drag repositioning, `Escape` close, outside-click close, and document immutability.

## Steps
- Open a markdown document containing an image (`![alt](url)`).
- Switch to WYSIWYG or Reading mode so the image renders.
- Double-click the image.
- Confirm a floating window appears with the image at full (or natural) size.
- Drag the window by its title bar to a new position and confirm it moves.
- Press Escape or click outside the window and confirm it closes.
- Confirm no editor content was modified.
