# MF-047 - Paragraph shortcuts convert the current line between heading levels 1-6 and plain paragraph text

## Notes
Typora documents paragraph shortcuts for Heading 1-6 (`Cmd`/`Ctrl`+`1..6`) and Paragraph (`Cmd`/`Ctrl`+`0`), and its Markdown Reference calls out typing heading markers or using the shortcut family as first-class authoring behavior. MarkFlow now handles these shortcuts through `packages/editor/src/editor/extensions/smartInput.ts`, which rewrites only the active line's markdown source while preserving the rest of the document unchanged; regression coverage in `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx` verifies paragraph-to-heading conversion, heading-to-paragraph reset, and active-line-only edits. Verified on 2026-04-11 with `pnpm --filter @markflow/editor test:run -- src/editor/__tests__/MarkFlowEditor.test.tsx` plus `node scripts/harness/verify.mjs`. Sources: https://support.typora.io/Shortcut-Keys/ and https://support.typora.io/Markdown-Reference/.

## Steps
- Place the caret on a plain paragraph line and press `Cmd`/`Ctrl`+`1` through `6` to apply the matching heading level.
- Place the caret on an existing heading line and press `Cmd`/`Ctrl`+`0` to convert it back to a plain paragraph.
- Confirm the current line changes in-place, other lines remain untouched, and the underlying markdown source stays editable.
