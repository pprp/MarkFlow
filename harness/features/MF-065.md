# MF-065 - Paragraph shortcuts convert the current line into quote, ordered-list, and unordered-list blocks

## Notes
Verified on 2026-04-12. Typora's Quote (`Ctrl+Shift+Q` / `Cmd+Option+Q`), Ordered List (`Ctrl+Shift+[` / `Cmd+Option+O`), Unordered List (`Ctrl+Shift+]` / `Cmd+Option+U`), and Paragraph (`Mod-0`) shortcuts all work correctly in `packages/editor/src/editor/extensions/smartInput.ts`. Quote/list toggles on active line, Mod-0 strips prefixes back to plain text, task-list lines preserved. Coverage in `MarkFlowEditor.test.tsx` and `smartInput.test.ts`; 231 tests pass. Not-tested: manual desktop shortcut check in Electron shell.

## Steps
- Place the caret on a plain paragraph line and press Typora's Quote shortcut (`Ctrl`+`Shift`+`Q` on Windows/Linux, `Cmd`+`Option`+`Q` on macOS), then confirm only the active line rewrites to `> ` markdown.
- Repeat with Typora's Ordered List shortcut (`Ctrl`+`Shift`+`[` / `Cmd`+`Option`+`O`) and Unordered List shortcut (`Ctrl`+`Shift`+`]` / `Cmd`+`Option`+`U`), confirming the active line rewrites in place to `1. ` and `- ` style list markup without mutating adjacent lines.
- Press Typora's Paragraph shortcut (`Cmd`/`Ctrl`+`0`) on each converted line and confirm MarkFlow returns that line to plain paragraph text while preserving surrounding content and caret position.
