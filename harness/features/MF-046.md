# MF-046 - Export pipeline integrates with Pandoc to generate DOCX, EPUB, and LaTeX artifacts from the active document

## Notes
Typora relies on Pandoc as a bridge for exporting to advanced formats like Word (.docx), EPUB, LaTeX, and OpenOffice. Implementation complete: `FileManager.exportPandoc()` in packages/desktop/src/main/fileManager.ts handles CLI invocation with correct args (--standalone for LaTeX), IPC handlers wired in registerIpcHandlers(), preload exposes exportDocx/exportEpub/exportLatex, menu items trigger export-docx/epub/latex actions, and App.tsx handlePandocExport() routes these to the API. Fixed in this run: electron mock missing `app` in fileManager.test.ts (blocking smoke), and missing handlePandocExport function + payload.action reference error in App.tsx. Sources: https://support.typora.io/Export/

## Steps
- Ensure Pandoc is installed on the host system or configured in MarkFlow.
- Open a markdown document with headings, lists, code, and links.
- Export the document to DOCX, EPUB, and LaTeX formats.
- Confirm the exported artifacts preserve document structure and styling.
