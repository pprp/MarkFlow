# MF-152 - Snippet and block template library inserts reusable writing structures from the command palette

## Notes
Reusable writing structures help MarkFlow feel fast without adding heavy machinery. This feature adds a small template library for common blocks such as meeting notes, ADRs, PRDs, experiment logs, paper summaries, code explanations, and weekly updates, exposed through the command palette and normal insert actions.

## Steps
- Define a local snippet/template registry with built-in templates and room for user-defined entries later.
- Expose template insertion through the command palette and Insert menu.
- Insert templates at the current selection, replacing selected text when appropriate and preserving undo history.
- Support simple placeholders such as title, date, selected text, and cursor landing position.

## Automated Verification
- Add registry tests for built-in template metadata and placeholder expansion.
- Add editor command tests proving insertion preserves undo history and selection behavior.
- Add command-palette tests proving templates are searchable and invokable by name.

## Manual Verification
- Insert each built-in template from the command palette into a new document and confirm placeholders, cursor position, and undo behavior feel natural.
