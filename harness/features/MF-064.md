# MF-064 - Increase/Decrease Heading Level shortcuts promote or demote the current heading by one level

## Notes
Typora's Shortcut Keys doc lists `Increase Heading Level` (`Ctrl`/`Cmd`+`=`) and `Decrease Heading Level` (`Ctrl`/`Cmd`+`-`) under Paragraph, and MarkFlow now rewrites only the active heading line through the existing smart-input heading shortcut path, including the H1->paragraph and H6 cap boundaries. Automated coverage lives in `packages/editor/src/editor/__tests__/MarkFlowEditor.test.tsx`. Sources: https://support.typora.io/Shortcut-Keys/ and https://support.typora.io/Markdown-Reference/.

## Steps
- Place the caret on a heading line such as `## Heading` and press `Cmd`/`Ctrl`+`=`.
- Confirm the markdown source changes by one heading level in the deeper direction without changing the heading text or surrounding lines.
- Press `Cmd`/`Ctrl`+`-` on a heading and confirm it changes by one level in the shallower direction, with boundary cases at H1 and H6 handled safely.

## Manual Verification
