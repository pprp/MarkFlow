# MF-018 - Main branch merges produce installable desktop beta builds automatically

## Notes
CI workflow at .github/workflows/ci.yml runs lint (pnpm -r run lint), test (pnpm test — all packages), and build (pnpm build — full workspace) on every push and PR to main. The release-beta job runs after all checks pass on main push, builds the Electron app via electron-builder on macOS, and publishes via GH_TOKEN. electron-builder.yml is configured with DMG+ZIP targets for x64 and arm64. Also fixed in this run: TS4094 error in markdownPostProcessor.ts caused by private members on exported anonymous class type — renamed to named class with _ prefix convention.

## Steps
- Configure CI to build editor and desktop packages on `main`.
- Publish beta artifacts for supported platforms.
- Confirm the release notes and update channel match the published build.
